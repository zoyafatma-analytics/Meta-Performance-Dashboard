# 📊 Meta Ad Performance Dashboard

## 📝 Project Description

This is a **Meta Ad Performance Dashboard** developed using Power BI to track and analyze the effectiveness of advertising campaigns across Facebook and Instagram.

The dashboard provides a complete marketing funnel view — from awareness to engagement to conversions — through KPIs such as impressions, clicks, engagements, conversions, purchases, and budget utilization.

It also includes demographic, geographic, and time-based insights to support data-driven decision-making for marketing teams.

---

# 🎯 Business Objective

The business requires a centralized performance tracking dashboard for advertising campaigns running on Meta platforms such as Facebook and Instagram.

This dashboard helps the marketing team to:

- Identify the most effective platform (Facebook vs Instagram)
- Track campaign performance and ROI
- Optimize advertising budget allocation
- Monitor engagement and conversion trends
- Understand audience behavior and engagement patterns
- Analyze demographic and geographic performance

# ✨ Features & Highlights

## 1️⃣ Target Gender Analysis – Donut Chart
- Visualizes ad performance split by target gender
- Helps identify audience engagement patterns

## 2️⃣ Target Age Group Analysis – Bar Chart
- Displays engagement performance across age groups
- Helps optimize campaign targeting

## 3️⃣ Country-wise Performance – Map Visualization
- Shows campaign performance by country
- Helps identify high-performing geographic regions

## 4️⃣ Calendar Month Analysis – Calendar Heat Map
- Visualizes monthly performance trends
- Detects seasonal engagement patterns

## 5️⃣ Weekly Performance Trend – Stacked Column Chart
- Displays weekly trends categorized by ad type
- Compares campaign performance over time

## 6️⃣ Hourly Trend Analysis – Area Chart
- Shows activity distribution by hour of the day
- Helps identify peak engagement hours

## 7️⃣ Ad Type Performance – Matrix Visualization
- Compares KPIs across different ad types
- Evaluates Facebook vs Instagram performance
